# Mercedes Cuesta

Landing estática responsive para Mercedes Cuesta, dietista online.

## Stack

- HTML
- CSS
- JavaScript vanilla
- Sin frameworks
- Sin cookies analíticas/publicitarias
- Sin formularios propios
- Preparada para deploy en Vercel

## Estructura


## Antes de publicar

1. Cambiar todas las URLs `https://mercedes-cuesta-dietista.vercel.app/` por el dominio final si cambia.
2. Revisar `pages/aviso-legal.html`, `pages/privacidad.html` y `pages/cookies.html`.
3. Probar la web en móvil, tablet y escritorio.
4. Validar accesibilidad con Lighthouse.
5. Probar la imagen de Open Graph al compartir el enlace.
